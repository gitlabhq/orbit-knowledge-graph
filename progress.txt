## Identified Gaps

1. **CJS cross-file resolution (express, fastify)**: 0 cross-file edges despite CJS resolver wiring.
   Root cause: OXC's `common_js_require()` may not recognize all require patterns in pure CJS files.
   Impact: express (392 CJS imports, 0 resolved), fastify (1,216 CJS imports, 0 resolved)

2. **Vue Options API this.method()**: GitLab has 4,480 Vue files but only 104 CALLS from them.
   Root cause: Options API methods are object literal properties, not class methods.
   Phase 2 only walks ancestors looking for Class nodes, not object literals.
   Impact: ~4,000 Vue files with near-zero call graph coverage.

3. **Prisma failed to index**: Empty results. Needs investigation.

4. **1,406 Vue files with zero definitions**: 31% of GitLab Vue files.
   Possible cause: template-only files or script extraction issues.
